# Predicting Life Expectancy Using WHO Data

## Objective
Identify key socio-economic and health-related predictors of life expectancy
using regression analysis.

## Dataset
- World Health Organization life expectancy dataset
- Observations: 180+ countries over 15 years (2000-2015)
- Features include mortality rates, economic indicators, education, disease rates, and population health indicators
  
## Methods
- Simple Linear Regression (SLR)
- Multiple Linear Regression (MLR)
- Logistic Regression (LR)
- Model comparison using goodness-of-fit metrics

## Key Insights
- Income Composition of Resources was the strongest predictor of life expectancy
- Multivariate models significantly outperformed single-variable models
- Residual analysis confirmed model assumptions for selected models
